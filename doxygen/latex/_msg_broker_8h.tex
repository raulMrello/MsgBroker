class \hyperlink{class_msg_broker}{Msg\-Broker}
\begin{DoxyCompactList}\small\item\em This class provides publish/subscription infrastructure as interprocess communication mechanism. Topics are the exchanged data structures which by one side will be ready to be subscribed to, or updated by publish functions. It is a thread safe implementation, so it can be used in baremetal or R\-T\-O\-S-\/based applications. If a mutex is required on R\-T\-O\-S-\/based applications, def macro M\-U\-T\-E\-X\-\_\-\-L\-O\-C\-K and M\-U\-T\-E\-X\-\_\-\-U\-N\-L\-O\-C\-K can be adapted according with your needs. \end{DoxyCompactList}\item

\begin{DoxyCompactList}\small\item\em \hyperlink{struct_msg_broker_1_1_observer}{Observer} structure is formed by a handler and a notification callback.
