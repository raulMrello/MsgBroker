Detailed Description

This class provides publish/subscription infrastructure as interprocess communication mechanism. Topics are the exchanged data structures which by one side will be ready to be subscribed to, or updated by publish functions. It is a thread safe implementation, so it can be used in baremetal or RTOS-based applications. If a mutex is required on RTOS-based applications, def macro MUTEX_LOCK and MUTEX_UNLOCK can be adapted according with your needs.
